# Changelog

## 0.5.0 - 2026-07-12

- 增加 MatterSim-5M 内聚能和 CrystalNN 平均配位数受控 Worker；
- 增加上传结构的完整快速 SBR 判断接口与浏览器入口；
- 以 BaCrSi4O10 完成真实 HTTP 上传的 PTE/NTE 判定验证。
- 增加受控 MatterSim 弹性张量与 QHA 精确任务、任务状态查询、Web 提交与轮询入口；
- 增加仅 QHA 恢复、嵌套恢复链路、弹性证据复用与 QHA 位移级进度；
- 增加弹性正定性、未应变弹性能和虚频质量警告，以及已完成结果的重解析接口；
- 增加温区 `alpha(T)` ZTE ROM 与体积分数/质量分数换算；
- 增加受控 Agent 白名单工具调用与严格科学请求对话入口；
- 完成 6701 条数据库、SBR、快速筛选、ZTE ROM、Agent 与精确任务界面的实际浏览器验证。

## 0.4.0 - 2026-07-12

- 增加受控外部ALIGNN Worker，隔离旧版Torch、DGL和JARVIS依赖；
- 增加上传CIF/POSCAR后真实预测剪切模量G的API和界面入口；
- 以BaCrSi4O10结构完成真实模型、HTTP上传接口和CPU回退验证。

## 0.3.0 - 2026-07-12

- 增加6701条材料检索、详情、G-Ẽ景观与数据集状态接口；
- 增加SBR、快速SBR、ROM、CIF/POSCAR基础检查接口；
- 增加Web/API端到端测试、PowerShell启动脚本和响应式界面优化。

## 0.2.0 - 2026-07-12
