# Changelog

## Unreleased

- 材料数据库增加 G、Ẽ、ξ、CTE 全库排序，CTE 区间筛选和 50/100/200 条显示控制；
- 增加最多 4 个材料的本地持久化收藏，支持属性表和真实 QHA α(T) 曲线叠加对比；
- 材料详情增加数据版本、源文件 SHA256、ALIGNN 剪切模量说明、Ẽ 公式和 QHA 曲线来源；
- Agent 增加 `compare_catalog_materials` 工具，可直接比较指定材料在任意温度的真实曲线插值结果；
- 将结构预测工作区标记为实验性扩展，明确当前版本以数据库查询和分析为主。

## 0.10.0 - 2026-07-14

- 按论文定义统一键合模量 `E_tilde = U_V/n = 160.21766208*abs(E_coh)/(AAV*avg_cn)`，数据库列表、详情、景观和Agent不再使用不一致的历史字段；
- 源码启动缺少目录库时，可从公开GitHub Release自动下载便携包、验证双重SHA256并原子安装SQLite目录库；
- 增加Windows、macOS、Linux三系统GitHub Actions，自动安装公开目录库、运行测试和验证无密钥便携包；
- 增加“关于软件”工作区，集中展示软件版本、著作权人/开发单位、数据版本、科学定义、技术组成和源码仓库；
- 窄屏设备默认收起右下角Agent，避免对话窗口遮挡数据库和关于页面主体内容；
- 统一Python包、FastAPI、网页静态资源和便携包版本为0.10.0。

## 0.9.0 - 2026-07-13

- 增加 Windows、macOS、Linux 共用的项目内便携运行环境和启动器；
- 增加默认无密钥发布包与显式合作者私有包构建流程；
- 移除源码中的开发机计算绝对路径，改用项目内可选 `compute.env`；
- 材料详情读取数据库内真实 POSCAR，使用 pymatgen CrystalNN 构建跨晶胞周期键合图，并由本地 3Dmol.js 显示可旋转、缩放和平移的三维晶体结构；
- 增加球棍、空间填充、键线、晶胞、周期邻居、超胞、视角重置、全屏和图片保存控制；
- 参考 Materials Project 的 VESTA 配色、统一原子半径、周期镜像和竖向晶胞表达，并将三维结构与真实 QHA 热膨胀曲线等宽等高并排展示；
- 将固定 CSS 坐标轴替换为独立 WebGL 三维坐标轴，仅同步主晶体的旋转四元数，并支持随重置视角恢复；
- 将三维坐标轴的 X/Y/Z 改为高清 HTML 标签，保持与 WebGL 箭头端点同步旋转并改善小尺寸显示清晰度；
- Fig. 1d 散点与当前材料星标按绘图区尺寸保持固定视觉比例平滑缩放，并同步调整悬停命中范围；
- 参考 Materials Project 重构数据库入口：将四个统计卡压缩为单条数据库摘要，并增加可折叠元素周期表；
- 元素周期表显示每种元素在6701个材料中的覆盖数量，支持“包含所选元素”和“仅含所选元素”的真实 SQLite 联合筛选；
- 将单页长页面拆分为材料数据库、结构预测、热膨胀景观和 ZTE 复合设计四个独立工作区路由，并保留全局 Agent；
- 元素周期表改为材料数据库工作区的常驻筛选组件，取消展开开关；
- 增加跨工作区“当前研究材料”上下文条、景观定位入口和状态恢复，让数据库材料及结构预测结果在多页面间保持连续分析体验；
- 三维渲染库和许可证随便携包离线分发，pymatgen/CrystalNN 作为项目内 Python 环境依赖安装。

## 0.8.0 - 2026-07-12

- 将快速预测改为 `fast_structure_screening` 后台任务，避免阻塞 Agent 对话；
- 增加统一的 `request_structure_calculation` 工具，由模型在 fast、elastic、qha 中自主选择；
- 三种计算均复用持久化审批协议，未经用户确认不会执行；
- 对话任务卡片可分别展示快速 SBR、精准弹性结果或完整 QHA 曲线；
- 保留旧 QHA 工具作为兼容入口，并新增快速与弹性审批路由测试。

## 0.7.0 - 2026-07-12

- 参考 `openai/codex` 的工具路由与集中审批边界，增加持久化 Agent 动作请求；
- 支持在 Agent 对话窗口上传 CIF/POSCAR，并将结构 ID 注入模型上下文；
- 增加结构检查、QHA 审批请求和计算任务状态通用工具；
- QHA 在用户点击“确认并提交”前不会启动，支持拒绝和重复审批保护；
- 审批后复用现有后台任务系统，持续显示进度并在完成后绘制完整 α(T) 曲线；
- 真实模型验证能够自主执行“检查结构 → 创建 QHA 审批请求”，且不会绕过确认。

## 0.6.0 - 2026-07-12

- 明确项目内 `var/` 为便携数据目录，不固定占用用户 C 盘；
- 接入 OpenAI 兼容的 `gpt-5.6-lunar` Agent，并支持多轮白名单工具调用；
- Agent 可检索 NTE/PTE 目录、读取材料与 QHA 曲线、计算 SBR 和设计曲线 ZTE 配比；
- 增加项目内本地环境变量配置文件，密钥文件不进入 Git、网页或日志；
- 增加 Agent 能力状态接口、网页配置提示和模型工具调用测试。
- 将 Agent 提升为右下角悬浮对话窗口，支持最小化、消息气泡和页面内多轮上下文。
- 增加任意温度真实 QHA 曲线的通用全库筛选与排名工具，并强化 Agent 自主规划规则。
