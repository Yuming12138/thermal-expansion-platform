# 软件著作权交付说明

软件名称：热膨胀材料智能计算与设计一体化平台 V1.0。

## 核心功能

1. 6701条热膨胀候选材料的版本化数据库、检索、详情、质量标记与可交互三维晶体结构显示。
2. CIF/POSCAR结构上传、ALIGNN剪切模量预测、MatterSim内聚能、CrystalNN配位数和快速SBR判别。
3. MatterSim弹性张量与QHA精确计算任务的受控提交、状态、日志和结果质量检查。
4. 单温点与温区alpha(T)曲线的ZTE复合配比设计，支持体积分数与质量分数。
5. Web材料景观、交互计算表单和受控Agent对话入口。

## 科学与安全边界

- 快速判断是筛选结果，边界样品必须使用完整弹性张量和QHA确认。
- 精确工作流只允许固定的WSL、Conda、MatterSim、VASPKIT和Phonopy命令组合。
- 若声子计算存在虚频，平台使用 Phonopy 的诊断性绝对频率近似生成热性质，并将结果标记为定性参考，不能替代动力学稳定性确认。
- Agent只调用注册工具，不执行Shell、文件删除或任意用户命令。
